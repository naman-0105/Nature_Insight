Welcome to Nature Insight, a user-friendly website designed to help you explore and learn about the fascinating world of plants. By simply uploading an image, our advanced system, powered by the PlantNet API, identifies the plant species and provides detailed information, including its scientific name, habitat, and care instructions. 4. **Start The Application**
```sh 
node server.js
```
